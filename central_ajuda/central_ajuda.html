Antes de iniciar o formulário

Confira os dados e documentos normalmente necessários para simular o pedido de habilitação de casamento no SIGACRC.

Dados dos contraentes

  • Nome completo, CPF, RG e data de nascimento.
  • Estado civil, profissão, nacionalidade e naturalidade.
  • Endereço completo com CEP de Osasco/SP.

Testemunhas

  • Duas testemunhas maiores de 18 anos.
  • Nome completo, RG, profissão e endereço.
  • Documentos de identificação anexados no final.

Documentos

  • Certidões atualizadas.
  • RG, CPF ou CNH dos noivos.
  • RG ou CNH das testemunhas.
  • Comprovante de residência.

Observação acadêmica

Esta versão simula a tramitação dos pedidos usando armazenamento local do navegador. Em uma versão final de produção, os dados seriam enviados para um banco de dados com autenticação, permissões e segurança adequada.
